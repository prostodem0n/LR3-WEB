Теги для роботи з текстом:

Тег b

HTML-елемент є частиною тексту, що стилістично відрізняється від нормального тексту, що не носить будь-якого спеціального значення.

Тег em

HTML елемент відзначає акцентований текст.

Тег i

HTML тег використовується для тексту, який з різних причин відрізняється від звичайного тексту.

Тег small

HTML тег визначає текст котрий являє собою юридичну формальність - щось незначне

Тег strong

Елемент сильної значущості вказує на те, що його зміст має велике значення, серйозність чи терміновість.

Тег sub

C2H5OH

Тег sup

C2 = A2 - B 2

Теги del та ins

Рекомендують 10/10 9/10 стоматологів!

Тег mark

HTML елемент що представляє текст, виділений у довідкових цілях через свою актуальність у певному контексті.

Тег abbr

Абревіатура АКБ.

Тег address

ДТЕУ знаходиться за адресою:

м. Київ, вул. Кіото, 19

Тег bdo

(зліва на право) Козак з казок

(зправа на ліво) Козак з казок

Тег blockquote

Елемент HTML вказує на те, що вкладений текст є розширеною цитатою. Зазвичай це відображається візуально за допомогою відступу.

Тег cite

"IwbnfouevcmsiehwvudbqvGovernment"

Winston Churchill

Тег q

Елемент HTML вказує на те, що вкладений текст є короткою вбудованою цитатою. Більшість сучасних браузерів реалізують це, беручи текст у лапки.
